## 🚀 客户使用流程

### 第一次使用

1. **访问管理页面**
   - 打开浏览器
   - 访问：`https://your-domain.vercel.app/zh/admin/login`
   - 输入管理员密码
   - 点击"登录"

2. **开始编辑**
   - 登录后进入编辑界面
   - 修改职位信息、联系方式等
   - 点击"发布"保存

3. **查看效果**
   - 访问前端页面：`https://your-domain.vercel.app/zh/careers`
   - 等待 30 秒自动更新，或手动刷新（F5）

### 日常使用

1. 访问管理页面 → 登录
2. 编辑内容 → 发布
3. 查看前端页面 → 自动更新

## 🔒 安全说明

### 已实现的安全功能

1. **密码保护**
   - 管理页面需要密码才能访问
   - 密码存储在 Vercel 环境变量中（安全）

2. **API 保护**
   - 所有编辑操作都需要验证密码
   - 未授权用户无法修改数据

3. **会话管理**
   - 30 分钟无操作自动登出
   - 防止未授权访问

### 安全建议

1. **定期更换密码**
   - 建议每 3-6 个月更换一次
   - 在 Vercel 环境变量中更新

2. **密码强度**
   - 至少 12 位字符
   - 包含大小写字母、数字、符号

3. **访问控制**
   - 只将密码提供给授权人员
   - 不要将密码写在公开文档中

## 🔧 维护说明

### 如果需要修改密码

1. 进入 Vercel Dashboard
2. 项目设置 → Environment Variables
3. 找到 `ADMIN_PASSWORD`
4. 编辑并保存
5. 重新部署项目（或等待自动部署）

### 如果需要查看日志

1. 进入 Vercel Dashboard
2. 项目 → Deployments
3. 选择最新部署
4. 查看 Functions 日志

### 如果需要备份数据

数据存储在 Vercel KV 中，Vercel 会自动备份。也可以：
1. 访问管理页面
2. 导出数据（可以添加此功能）
3. 或直接从 Vercel KV 导出

## 📞 技术支持

### 客户遇到问题时

1. **无法登录**
   - 检查密码是否正确
   - 检查网络连接
   - 清除浏览器缓存

2. **编辑后没有更新**
   - 确认是否点击了"发布"按钮
   - 等待 30 秒自动更新
   - 或手动刷新前端页面

3. **其他问题**
   - 查看浏览器控制台错误（F12）
   - 检查 Vercel 部署日志
   - 联系技术支持
