# Sanity Studio 客户使用指南

## 📖 简介

本指南面向客户，说明如何在 Sanity Studio 中编辑招聘信息。

## 🌐 访问 Sanity Studio

### 访问地址

在浏览器中打开以下地址：

```
https://your-domain.vercel.app/studio
```

### 首次访问

1. 打开上述地址
2. 如果提示登录，使用您的 Sanity 账号登录
   - 如果没有账号，需要先注册 Sanity 账号
   - 或联系技术人员为您创建账号

## ✏️ 编辑招聘信息

### 进入编辑界面

1. 登录后，您会看到 Sanity Studio 界面
2. 在左侧菜单中找到 **"招聘页面"**（Careers Page）
3. 点击进入编辑

### 编辑职位信息

#### 添加新职位

1. 在 **"招聘职位"**（Jobs）部分
2. 点击 **"Add item"** 或 **"添加项"**
3. 填写职位信息：
   - **职位 ID**：输入唯一标识（如：`business`、`translator`、`freight`）
   - **职位标题**：填写中英文标题
   - **薪资待遇**：填写中英文薪资说明
   - **核心职责**：每行一个条目
   - **职位要求**：每行一个条目
   - **优先条件**：每行一个条目（可选）

#### 编辑现有职位

1. 找到要编辑的职位
2. 点击职位卡片
3. 修改相应字段
4. 修改完成后，点击右上角 **"Publish"**（发布）按钮

#### 删除职位

1. 找到要删除的职位
2. 点击职位卡片
3. 点击右上角的删除按钮
4. 确认删除

### 编辑联系方式

1. 在 **"联系方式"**（Contact）部分
2. 修改以下信息：
   - **电话**（Phone）
   - **邮箱**（Email）
   - **地址**（Address）- 中英文

## 💾 保存和发布

### 重要：必须发布

- **编辑内容**：只是修改，不会更新网站
- **发布（Publish）**：才会真正更新网站

### 发布步骤

1. 编辑完成后，点击右上角的 **"Publish"** 按钮
2. 确认发布
3. 等待几秒钟
4. 看到 **"Published"** 提示表示成功

### 自动部署

发布后：
1. 系统会自动触发 Vercel 重新部署
2. 通常需要 2-5 分钟
3. 部署完成后，前端页面会自动显示最新内容

## 🔄 查看更新效果

### 方法 1: 等待自动更新

- 发布后等待 2-5 分钟
- 前端页面会自动更新

### 方法 2: 手动查看

1. 打开前端页面：
   ```
   https://your-domain.vercel.app/zh/careers
   ```
2. 按 `F5` 刷新页面
3. 查看最新内容

## 📝 编辑提示

### 文本格式

- **核心职责、职位要求、优先条件**：
  - 每行一个条目
  - 直接输入文本即可
  - 不需要特殊格式

### 中英文编辑

- 所有字段都支持中英文
- 确保两种语言都填写完整
- 可以只填写一种语言（但建议都填写）

### 职位 ID

- 必须是唯一的
- 建议使用英文和数字
- 例如：`business`、`translator`、`freight`、`job-001`

## ⚠️ 注意事项

### 1. 必须发布

- 编辑后必须点击 **"Publish"** 才会生效
- 只编辑不发布，网站不会更新

### 2. 自动部署时间

- 发布后需要等待 2-5 分钟
- 这是正常的，请耐心等待

### 3. 浏览器兼容性

- 推荐使用 Chrome、Edge、Firefox
- 不建议使用 IE 浏览器

### 4. 网络连接

- 需要稳定的网络连接
- 如果编辑失败，检查网络后重试

## 🆘 常见问题

### Q: 找不到"招聘页面"？

**A**: 
1. 检查是否已登录
2. 刷新页面（F5）
3. 联系技术支持

### Q: 发布后网站没有更新？

**A**: 
1. 确认是否点击了 "Publish" 按钮
2. 等待 2-5 分钟（自动部署需要时间）
3. 清除浏览器缓存后重试
4. 联系技术支持检查部署状态

### Q: 可以撤销修改吗？

**A**: 
- Sanity 有版本历史功能
- 可以查看之前的版本并恢复
- 在编辑界面右上角找到历史记录

### Q: 可以多人同时编辑吗？

**A**: 
- 可以，但建议不要同时编辑同一个职位
- 最后保存的内容会覆盖之前的修改
