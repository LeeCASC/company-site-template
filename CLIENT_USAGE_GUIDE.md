# 客户使用指南

## 📖 简介

本指南面向不熟悉技术的客户，说明如何在线编辑和管理招聘信息。

## 🌐 访问方式

### 管理页面（编辑内容）

访问以下地址进行编辑：

**中文版**：
```
https://your-domain.vercel.app/zh/admin/login
```

**英文版**：
```
https://your-domain.vercel.app/en/admin/login
```

### 前端页面（查看效果）

编辑完成后，访问以下地址查看效果：

**中文版**：
```
https://your-domain.vercel.app/zh/careers
或
https://your-domain.vercel.app/careers.html
```

**英文版**：
```
https://your-domain.vercel.app/en/careers
```

## 🔐 登录

1. 打开管理页面地址
2. 输入管理员密码（由技术人员提供）
3. 点击"登录"按钮
4. 登录成功后进入编辑界面

## ✏️ 编辑内容

### 基本操作

1. **编辑职位信息**
   - 找到要编辑的职位
   - 直接在输入框中修改内容
   - 支持中英文双语编辑

2. **添加新职位**
   - 点击"添加新职位"按钮
   - 填写职位信息
   - 保存即可

3. **删除职位**
   - 找到要删除的职位
   - 点击职位右上角的"−"按钮
   - 确认删除

4. **编辑联系方式**
   - 在页面底部的"联系方式"部分
   - 修改电话、邮箱、地址等信息

### 编辑字段说明

- **职位标题**：职位的名称（中英文）
- **薪资文案**：薪资待遇说明（中英文）
- **核心职责**：职位的主要工作内容
  - 每行一个条目
  - 支持纯文本或 HTML 格式
- **职位要求**：应聘者需要满足的条件
  - 每行一个条目
  - 支持纯文本或 HTML 格式
- **优先条件**：加分项（可选）
  - 每行一个条目

## 💾 保存和发布

### 保存草稿

- 点击"保存草稿（本地）"按钮
- 数据保存在浏览器中，不会更新网站
- 适合临时保存，防止数据丢失

### 发布

1. 编辑完成后，点击"发布"按钮
2. 等待几秒钟（系统正在保存）
3. 看到"已发布成功"提示
4. **重要**：前端页面会在 30 秒内自动更新
5. 也可以手动刷新前端页面查看效果

## 🔄 自动更新机制

### 前端页面自动更新

- 前端页面每 30 秒自动检查一次更新
- 如果检测到内容有变化，会自动刷新显示
- **无需手动操作**

### 立即查看更新

如果想立即查看更新效果：
1. 发布后等待 5-10 秒
2. 打开前端页面（`/zh/careers` 或 `/careers.html`）
3. 按 `F5` 或点击浏览器刷新按钮
4. 即可看到最新内容

## ⚠️ 注意事项

### 1. 会话过期

- 登录后 30 分钟无操作会自动登出
- 如果被登出，重新登录即可

### 2. 数据保存

- **保存草稿**：只保存在浏览器，不会更新网站
- **发布**：才会真正更新网站内容
- 建议编辑完成后立即发布

### 3. 格式要求

- **核心职责、职位要求、优先条件**：
  - 每行一个条目
  - 可以直接输入文本，系统会自动处理
  - 也可以输入 HTML 代码（高级用户）

### 4. 浏览器兼容性

- 推荐使用 Chrome、Edge、Firefox 等现代浏览器
- 不建议使用 IE 浏览器

## 🆘 常见问题

### Q: 登录后看不到内容？

**A**: 
1. 检查网络连接
2. 刷新页面（F5）
3. 清除浏览器缓存后重试

### Q: 发布后前端页面没有更新？

**A**: 
1. 等待 30 秒（自动更新）
2. 或手动刷新前端页面（F5）
3. 清除浏览器缓存后重试

### Q: 忘记密码怎么办？

**A**: 联系技术人员重置密码

### Q: 可以多人同时编辑吗？

**A**: 
- 可以，但建议不要同时编辑同一个职位
- 最后保存的内容会覆盖之前的修改

### Q: 编辑的内容会丢失吗？

**A**: 
- 点击"保存草稿"可以防止数据丢失
- 发布后数据会保存到服务器，不会丢失

## 🎯 快速操作流程

1. **登录** → 访问管理页面，输入密码
2. **编辑** → 修改职位信息或联系方式
3. **发布** → 点击"发布"按钮
4. **查看** → 等待 30 秒或刷新前端页面查看效果
